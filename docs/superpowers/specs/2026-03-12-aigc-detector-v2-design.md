# AIGC 文本检测器 v2 - 设计规格

## 项目概述

- **项目名称**: AIGC 文本检测器 (文献阅读器风格)
- **类型**: Web 应用 (Flask + HTML/CSS/JS)
- **核心功能**: 带有多级别检测和荧光笔标注的文献阅读风格界面，支持文件导入和实时编辑修改
- **目标用户**: 需要降低 AI 写作率的作者、学生、教育工作者

## 功能规格

### 1. Word/TXT 文档解析

| 功能 | 说明 |
|------|------|
| TXT 导入 | 支持 UTF-8 编码的文本文件 |
| Word 导入 | 支持 .docx 文件，使用 python-docx，尝试多种编码避免乱码 |
| 编码修复 | python-docx 添加编码检测和多种编码尝试 |

### 2. 多级别分块检测

| 级别 | 说明 |
|------|------|
| 段落级 | 按换行符分块，每段独立检测 |
| 句子级 | 按句号/感叹号/问号分句，可展开查看 |
| 切换功能 | 用户可切换不同级别查看 |

### 3. 文献阅读器风格界面

**布局结构**:
```
┌─────────────────────────────────────────┬──────────────────┐
│  AIGC检测                         [导入] │                  │
├─────────────────────────────────────────┤                  │
│  [段落级] [句子级]                       │                  │
├─────────────────────────────────────────┼──────────────────┤
│                                         │  选中块详情      │
│  文本阅读区                              │                  │
│  (带荧光笔高亮的原文)                     │  AI率: 85%      │
│                                         │  ─────────────  │
│  ██████████████████高风险                │                  │
│  段落内容...                             │  [编辑区域]     │
│                                         │  (可编辑文本)   │
│  ████████████中风险                      │                  │
│  段落内容...                             │  (输入时自动    │
│                                         │   重新检测)     │
│  ████低风险                              │                  │
│  段落内容...                             │  ─────────────  │
│                                         │  整体AI率: 45% │
└─────────────────────────────────────────┴──────────────────┘
```

**荧光笔标注**:
- 高风险 (>50%): 红色背景 (#fee2e2)
- 中风险 (20-50%): 黄色背景 (#fef3c7)
- 低风险 (<20%): 绿色背景 (#dcfce7)

### 4. 交互功能

| 功能 | 说明 |
|------|------|
| 点击选中 | 点击任一段落/句子，在操作栏显示详情 |
| 自动检测 | 编辑文本后自动触发重新检测（防抖 500ms） |
| 整体统计 | 实时显示整体 AIGC 率 |

### 5. API 接口

```
POST /api/detect
  输入: { "text": "文本内容" }
  输出: { "success": true, "probability": 0.85 }

POST /api/detect-full
  输入: { "text": "长文本", "mode": "paragraph" | "sentence" }
  输出: {
    "success": true,
    "overall_probability": 0.45,
    "chunks": [
      { "text": "段落1", "probability": 0.85, "level": "paragraph" },
      { "text": "段落2", "probability": 0.30, "level": "paragraph" }
    ]
  }

POST /api/detect-chunk
  输入: { "text": "单段文本" }
  输出: { "success": true, "probability": 0.65 }

POST /api/upload
  输入: multipart/form-data (file)
  输出: { "success": true, "text": "提取的文本" }
```

### 设计特点
- 左侧原文阅读区，带荧光笔高亮
- 右侧操作栏，显示选中块详情
- 可编辑文本区域，编辑时自动检测
- 清晰的视觉层次

## 技术实现

### 后端 (Flask)
- 修复 python-docx 编码问题
- 添加 `/api/detect-full` 接口（多级别检测）
- 添加 `/api/detect-chunk` 接口（单块检测）

### 前端
- 双栏布局：阅读区 + 操作栏
- 荧光笔高亮效果
- 点击选中交互
- 编辑器自动检测（防抖）

4. ✅ 荧光笔高亮显示正确
5. ✅ 点击段落显示在操作栏
6. ✅ 编辑后自动重新检测
7. ✅ 整体 AIGC 率实时更新

# AIGC 文本检测器增强版 - 设计规格

## 项目概述

- **项目名称**: AIGC 文本检测器 (增强版)
- **类型**: Web 应用 (Flask + HTML/CSS/JS)
- **核心功能**: 对长文本进行整体和分段 AIGC 检测，支持文件导入，便于用户定位和修改高风险区域
- **目标用户**: 需要降低 AI 写作率的作者、学生、教育工作者

## 功能规格

### 1. 文本输入

| 功能 | 说明 |
|------|------|
| 粘贴输入 | 大文本输入框，支持直接粘贴 |
| TXT 导入 | 点击按钮选择本地 .txt 文件 |
| Word 导入 | 点击按钮选择本地 .docx 文件 |

### 2. 分段检测

| 功能 | 说明 |
|------|------|
| 块大小选择 | 用户可选: 100字, 200字, 500字, 1000字 |
| 智能分块 | 按句子/段落分割，保证句子完整 |
| 逐块检测 | 对每个块独立进行 AIGC 检测 |
| 整体检测 | 对全文进行整体 AIGC 率检测 |

### 3. 结果展示

| 功能 | 说明 |
|------|------|
| 整体概率 | 显示全文 AIGC 概率百分比 |
| 分块可视化 | 每个文本块显示独立概率条 |
| 风险标识 | 高风险(>50%)红色，中风险(20-50%)黄色，低风险(<20%)绿色 |
| 点击查看 | 点击任一块可查看该块详情 |

### 4. API 接口

```
POST /api/detect
  输入: { "text": "文本内容" }
  输出: { "success": true, "probability": 0.85, "result": "AI生成的文本", "text_length": 100 }

POST /api/detect-chunks
  输入: { "text": "长文本", "chunk_size": 200 }
  输出: {
    "success": true,
    "overall_probability": 0.65,
    "total_chunks": 5,
    "chunks": [
      { "text": "第一段...", "probability": 0.3, "start": 0, "end": 150 },
      { "text": "第二段...", "probability": 0.8, "start": 150, "end": 320 },
      ...
    ]
  }

POST /api/upload
  输入: multipart/form-data (file)
  输出: { "success": true, "text": "提取的文本内容" }
```

## 技术实现

### 后端 (Flask)

- `app.py` - 主应用，增加新接口
- `python-docx` - 解析 Word 文件
- 复用现有模型加载逻辑

### 前端

- `templates/index.html` - 全新设计
- Vanilla JS (无框架)
- 文件上传 (FormData API)
- 响应式布局

# 纯文本模式与 Word 模式切换设计

## 概述

为 AIGC 检测器添加纯文本模式作为默认入口，支持与 Word 模式切换。Word 模式因跳转功能存在 bug，切换时需弹出警告提示用户。

## 需求摘要

1. 默认进入纯文本模式（而非 Word 模式）
2. 左侧边栏添加「纯文本」「Word」切换按钮
3. 点击 Word 按钮时弹出警告，告知跳转功能 bug

## 设计方案

### 左侧边栏布局

```
┌──────────────┐
│   📄 打开文件  │
├──────────────┤
│   🔍 检测     │
├──────────────┤
│   📊 统计     │
├──────────────┤
│   📥 导出 ▼   │
├──────────────┤
│   📝 纯文本    │  ← 当前选中模式
│   📄 Word    │  ← 切换到 Word 模式
├──────────────┤
│   ☀️/🌙 主题   │
└──────────────┘
```

### 模式说明

| 模式 | 左侧编辑器 | 右侧预览 | 跳转功能 |
|------|-----------|----------|----------|
| **纯文本** | `<textarea>` 纯文本输入框 | 分块预览列表 | ✅ 正常（高亮定位） |
| **Word** | SuperDoc 编辑器 | 分块预览列表 | ❌ 有 bug（跳转无效） |

### Word 模式警告弹窗

用户点击 Word 按钮时显示：

```
┌─────────────────────────────────────────────┐
│              ⚠️ 提示                         │
├─────────────────────────────────────────────┤
│                                             │
│   Word 模式跳转功能正在修复中                  │
│                                             │
│   当前问题：点击右侧分块无法跳转到左侧          │
│   Word 编辑器的对应位置                       │
│                                             │
│   ┌─────────────────────────────────────┐   │
│   │  仍然切换到 Word 模式（编辑功能可用）   │   │
│   └─────────────────────────────────────┘   │
│                                             │
│   ┌─────────────────────────────────────┐   │
│   │  留在纯文本模式                       │   │
│   └─────────────────────────────────────┘   │
│                                             │
└─────────────────────────────────────────────┘
```

### 状态管理

新增 state：
- `editorMode: 'plaintext' | 'word'` — 当前编辑器模式，默认 `'plaintext'`

### 文件变更

**frontend/src/App.tsx**
- 新增模式切换按钮组件
- 新增 Word 模式警告弹窗组件
- 新增纯文本编辑器（textarea）UI
- 根据 `editorMode` 条件渲染编辑器类型

### 实现步骤

1. 创建模式切换按钮（纯文本/Word）
2. 创建警告弹窗组件
3. 添加纯文本模式的 textarea 编辑器 UI
4. 实现模式切换逻辑与弹窗交互
5. 更新样式支持纯文本模式布局

## 组件清单

| 组件 | 用途 |
|------|------|
| `ModeToggle` | 模式切换按钮组 |
| `WordWarningModal` | Word 模式警告弹窗 |

## 状态定义

```typescript
interface AppState {
  editorMode: 'plaintext' | 'word'  // 默认 'plaintext'
  showWordWarning: boolean            // 弹窗显示状态
}
```
